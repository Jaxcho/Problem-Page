Problem Page
- People can post problems using latex code
- Under each problem people can comment ideas
- When people post, they will have to input a correct solution that when you submit to that specific problem, it can tell you if it is right or wrong.
- People can bookmark problems, and store them under like "Problems to complete"/ "Problems to remember"
- People have accounts, logins, etc.
- People can rate problems, and that will be averaged.
- There will be sections for problems (Beginner, intermediate, advanced, proof) and under those (Alg, NM, Comb, Geo, Miscellaneous)
